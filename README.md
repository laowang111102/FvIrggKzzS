# 项目概述 python1015

本项目是一个基于Python的外卖配送分析与可视化系统，采用Django框架进行后端开发，结合Hive和爬虫技术进行数据处理，运用协同过滤算法进行智能推荐，并使用MySQL作为数据库管理系统，前端则采用Vue.js构建用户界面。

# 技术栈

- 后端：Python、Django
- 数据处理：Hive、爬虫
- 推荐算法：协同过滤算法
- 数据库：MySQL
- 前端：Vue.js
